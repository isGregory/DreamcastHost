This directory is where websites intending to be added to the server should be placed.
They should follow the example set by "Example".
The directory name is not important for installing a website, but should contain:
1. info.txt
	- This file specifies information such as the target website name.
2. site
	- This file specifies the apache2 settings the website will use.
3. public_html/
	- This directory contains the website files. Either html or php files.
